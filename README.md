GOLANG-WELCOMEPACK
==================
This is a repo showing a basic setup in GOLANG, using:

- [GO-TESTING](https://golang.org/pkg/testing/) for test
- ['STRETCHR-TESTIFY'](https://github.com/stretchr/testify) for assert


##Build
```
$ go build ./src/main/
```

##Test
```
$ go test ./src/main/
```

##Run
```
$ ./main
$ go run ./src/main/main.go
```
